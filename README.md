前排提示，使用和阅读本文需要读者具备：
1. 基本的中文阅读能力
2. 基本的Excel使用能力
3. 基本的网页浏览能力

本Repo目标：
1. 提供工具简化使用者个人资产情况的盘点和对未来时间节点的估算
2. 提供工具简化使用者对房贷的理解成本和估算成本
3. 提供工具简化使用者对新房，二手房交易流程的理解成本
4. 提供（个人）GDP回落时代的房产价值观点和购买方法论
5. 提供基于PR和Issue的北京板块介绍和点评
6. 提供基于季度更新的新盘介绍和点评
