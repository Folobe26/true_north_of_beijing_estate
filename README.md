# 置业因素


### 资产沉淀（保值增值）

作为第一份大额投资，保值增值属性肯定是第一位的，按照优先级排列如下。产业是第一支撑要素，有产业支撑的卫星城在后续的流通和保值能力上有非常大的
#### 产业配套

- 金融业

- 医疗行业

- 汽车/航天等尖端制造业
